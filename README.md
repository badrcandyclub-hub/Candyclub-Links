# Candy Club Hub

A premium, mobile-first, high-end digital hub and customer portal for "Candy Club".

## Project Overview

This project serves as a stunning visual experience that matches the brand's physical marketing materials, while offering distinct, functional customer service tools:
- Social media redirection
- Smart product request form
- Direct payment/contact links (WhatsApp, InstaPay)

## Technologies Used

- **HTML5**: Semantic and accessible structure.
- **CSS3 (Vanilla)**: Custom styling utilizing high-end Glassmorphism, CSS Variables, and smooth animations without external frameworks.
- **JavaScript (Vanilla)**: Form handling and interactive elements (like clipboard copy).

## Features

- **High-End Glassmorphism**: Frosted glass effects on cards and buttons with a pastel gradient background.
- **Arabic Typography & RTL**: Optimized for Arabic rendering (`dir="rtl"`) using Google Fonts (Cairo).
- **Responsive Design**: Mobile-first architecture ensuring flawless display on all devices.
- **Smart Form**: Interactive missing products request form with loading and success states.
- **InstaPay Integration**: Dedicated payment card with easy copy-to-clipboard functionality for the account number.
- **WhatsApp Integration**: Prominent call-to-action to contact support via WhatsApp.

## Setup

Simply open `index.html` in any modern web browser to view the application. No build tools or servers are required.
